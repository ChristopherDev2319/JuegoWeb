❌ Errores Comunes

Si ves estos errores, aquí están las causas:

1. "Cannot read properties of null (reading 'layers')"
   → Problema con Three.js o cámara no inicializada

2. "SyntaxError: Unexpected token"
   → Error de sintaxis en JavaScript

3. "Failed to load resource"
   → Archivos no encontrados (modelos, texturas, etc.)

4. "TypeError: Cannot read properties of undefined"
   → Variable no definida o módulo no cargado

5. "CORS policy" 
   → Usar servidor HTTP, no file://
